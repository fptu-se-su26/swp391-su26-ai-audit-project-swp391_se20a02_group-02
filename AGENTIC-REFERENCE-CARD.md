## 🤖 AGENTIC PATTERNS (6 Types)

| Pattern | What It Does | When to Use |
|---------|------------|-----------|
| **Reactive** | Input → Output | Simple predictions, fast |
| **Conversational** | Multi-turn dialogue | User interaction, discussion |
| **Autonomous** | Self-executing tasks | Auto-monitoring, alerts |
| **Tool-Using** | Chain multiple APIs | Complex operations |
| **Reasoning** | Explain decisions | Justification needed |
| **Multi-Agent** | Agents collaborate | Complex business strategy |
